Acerca de mí

Cuento con 10 años de experiencia enfocado en la mejora continua de procesos en la industria siderúrgica, ayudando a las empresas a obtener el máximo provechos de sus datos mediante la implementación de aplicaciones web para la gestión del mantenimiento y la toma de decisiones que aporten soluciones relacionadas a la productividad y a la confiabilidad de los equipos e instalaciones.

Desde mis inicios como ingeniero de mantenimiento me apoyé en el análisis de datos como herramienta fundamental para anticipar problemas y proponer soluciones. A lo largo de mi carrera profesional incorporé (y lo sigo haciendo) nuevas herramientas digitales para optimizar los flujos de trabajo y, de esta manera, poder brindar respuestas ágiles que contribuyen a la resolución de problemas del cliente. Algunas de estas herramientas son:

Python (Numpy, Pandas, Streamlit, Matplotlib, Plotly) | Power BI (DAX) | SQL | GIT | GITHUB | Docker | HTML5 | CSS3 | Probabilidad y Estadística | SAP PM | VBA | Storytelling
Me llama la atención todo lo relacionado a las nuevas tecnologías, como la IA, y su aplicación para resolver problemas del mundo real, así que espero que podamos contactar y trabajar juntos en este sentido.

Gestión de Taller de Mantenimiento con Power BI y SQL Server

Con esta herramienta, creada utilizando SQL Server y Power BI, se evalúa el desempeño de los equipos que componen la Máquina de Colada Continua (MCC) y que se reparan en el Taller de Mantenimiento. También se muestran las causas de fallas más frecuentes y las soluciones a estas problemáticas.
